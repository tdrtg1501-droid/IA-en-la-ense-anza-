# IA-en-la-ense-anza-
Web del TDR 'IA a l'Ensenyament'. Un projecte per transformar la IA en aliada docent. Accedeix al 'prompt' d'un assistent GPT dissenyat per personalitzar l'aprenentatge, reduir la burocràcia i fomentar el pensament crític, fugint de solucions automàtiques i prioritzant la pedagogia.
